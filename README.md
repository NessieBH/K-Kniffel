# K-Kniffel

Short version of Yahtzee for two players.
It counts: 
- Ones
- Three of a kind
- Small straight
- Yahtzee
- Chance
Have fun!

Dies ist eine Kurzversion von dem Würfelspiel Kniffel für zwei Spieler.
Es werden gezählt:
- Einser
- Dreierpasch
- Kleine Strasse
- Kniffel
- Chance
Viel Spass beim Spielen!
